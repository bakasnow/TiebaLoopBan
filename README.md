# 贴吧循环封禁

#### 介绍
软件版贴吧循环封禁，为服务器长期运行设计。

#### 界面
![软件预览图](https://www.bakasnow.com/tuchuang/tlb.png "tlb.png")

#### 功能
1. 每天凌晨5点起，自动封禁列表中的用户
2. 设定结束时间，避免无脑循环
3. 失败无限重试，防止漏封
4. 支持批量添加循封名单
